DB-GPT Implementation

Project Overview

DB-GPT is an AI-powered system that allows users to interact with databases using natural language queries. I was responsible for researching and implementing the core functionalities of DB-GPT, including integrating a GPT-3-based language model with various database management systems. This system simplifies data access by providing an intuitive interface for non-technical users to retrieve and manipulate data without SQL knowledge.

Technologies Used

  • GPT-3, FastAPI
  • PostgreSQL
  • Python, NLP, Docker
